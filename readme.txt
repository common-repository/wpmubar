== Description ==  

WpMuBar plugin provides you with easy to setup and use upper bar for your Multisite WordPress portal.

This free version of WPMUBar has some limitations.

Learn more about full version on 
http://store.theme.fm/plugins/wpmubar/

== Installation ==

Upload plugin to your WordPress site the way you are accustomed to. The simplest way is to upload plugin via WordPress admin panel (Plugins -> Add new -> Upload).
After successfull upload, check folder access rights.
Activate plugin via (Plugins -> Installed plugins -> Rosetta plugin -> Activate).

That’s all - we hope that you’ll like our plugin. Suggestions, questions and other feedback are welcomed. Email: support@theme.fm
